## Tipografía

El sitio utiliza la fuente **Poppins** de Google Fonts.

**Link de integración:**
```html
<link href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;600;700&display=swap" rel="stylesheet">
```

**Pesos utilizados:**
*   **300 (Light)**: Usado para subtítulos finos o detalles.
*   **400 (Regular)**: Usado para el cuerpo del texto general.
*   **600 (Semi-Bold)**: Usado para botones y énfasis moderado.
*   **700 (Bold)**: Usado para encabezados y títulos importantes.
